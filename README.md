# MultiTool - by Heiva

MultiTool est un script batch pour lancer rapidement des applications courantes comme Teams, Chrome, Roblox ou PuTTY.

## Exécution

1. Téléchargez ou clonez le dépôt.
2. Double-cliquez sur `MultiTool.bat`.
3. Le menu s'affichera avec votre bannière colorée.
4. Choisissez une option pour lancer l'application correspondante.

## Compatibilité

- Windows 10 et 11
- PowerShell 5+ requis pour le lancement des applications modernes
